# ☕ Интернет-магазин кофе "Кофейная лавка"

Полноценный веб-сайт для продажи кофе с личным кабинетом, корзиной, REST API и React-приложением. Проект выполнен в рамках дисциплин по клиент-серверным приложениям и сетевым технологиям.

## 🚀 О проекте

Проект представляет собой интернет-магазин по продаже кофе. Включает в себя:

- Статические страницы с каталогом товаров и карточками продуктов
- Систему регистрации и авторизации пользователей (PHP сессии)
- REST API для управления пользователями
- JavaScript-корзину с сохранением в `localStorage`
- React-приложение (визитка студента + загрузка данных из API)

## 🎯 Основной функционал

### Для всех пользователей
- **Каталог кофе** с фильтрацией по сортам (Арабика, Робуста, Либерика)
- **Карточки товаров** с подробным описанием и характеристиками
- **Корзина** (добавление/удаление товаров, подсчёт суммы, сохранение после перезагрузки)
- **Оплата** — уведомление о заказе и очистка корзины

### Для авторизованных пользователей
- **Личный кабинет** с приветствием и данными пользователя
- **Доступ к React-приложению** (визитка и список пользователей)

### REST API (доступно в `api/v1/`)
- `POST /register` — регистрация нового пользователя
- `POST /login` — авторизация
- `GET /users` — получение списка всех пользователей
- `GET /users/{id}` — получение одного пользователя
- `PUT /users/{id}` — обновление пароля
- `DELETE /users/{id}` — удаление пользователя

### React-приложение (ПР-7 и ПР-8)
- **ProfileCard** — визитка студента с загрузкой аватарки
- **UsersList + UserCard** — загрузка и отображение списка пользователей из тестового API
- **Состояния загрузки и ошибок** — отображение "Загрузка..." и "Ошибка загрузки данных"

## 🛠 Технологии

| Компонент       | Технологии |

| Фронтенд       | HTML5, CSS3, JavaScript (ES6) |
| Бэкенд         | PHP 7.4+ (нативная сессия, работа с JSON) |
| API            | REST API, JSON |
| React-приложение| React 18, Vite, Fetch API |
| Сборка React   | Vite |
| Хранение данных| JSON-файлы (`users.json`) |

## ⚙️ Установка и запуск

### Требования
- Установленный **XAMPP** (или любой другой сервер с PHP)
- **Node.js** (для разработки React-приложения)

### Инструкция

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/nfgone23/coffe-shop.git
2. **Переместите папку coffe-shop в htdocs (обычно C:/xampp/htdocs/)**
3. **Запустите XAMPP и включите Apache**
4. **Откройте сайт в браузере:**
   http://localhost/coffe-shop/
